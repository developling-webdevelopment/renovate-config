# Renovate developling-webdevelopment Configs

developling-webdevelopment presets for [Renovate](https://github.com/singapore/renovate) tool.

## Usage

Add this into `renovate.json`:

```json
{
  "extends": ["@developling-webdevelopment"]
}
```
